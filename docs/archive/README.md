# Archived — historical build notes

These documents record how the CMS was built, phase by phase. **They are not accurate descriptions
of the current system** and must not be used as reference. Several describe APIs that were removed
or slot names that were renamed — for example `return_delete_token` / `delete_by_token` (removed:
rejected by Cloudinary on unsigned uploads) and the old `service_mandap` / `about_portrait` slot
names (renamed when the 1:1 image mapping was fixed).

Kept for provenance only.
